# KPI Framework

## Core Product KPIs

| KPI | Definition | Target (Initial) |
|---|---|---|
| Processing Accuracy (%) | % of extracted elements accepted without manual correction | >= 93% |
| Time Saved per Document | Minutes reduced vs manual conversion | >= 60% reduction |
| Conversion Success Rate | % of uploads that complete end-to-end export | >= 98% |
| User Engagement | Weekly active users / invited users | >= 55% |
| Review Cycle Time | Time from upload to approved workflow output | < 20 minutes |
| Export Utilization | % of processed docs exported as JSON/Word/Markdown | >= 80% |

## Operational KPIs
- Median pipeline runtime per document.
- Low-confidence escalation rate.
- Error rate by stage (upload, parse, process, export).
